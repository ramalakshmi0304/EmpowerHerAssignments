# Portfolio Website
A simple, clean, responsive portfolio website with:

- Blue theme
- Navbar with theme toggle button
- Reduced hero/banner size
- Responsive skills grid
- No animations

## Files Included
- **index.html** — Structure of the website
- **style.css** — Styling and responsive layout
- **README.md** — Description and usage instructions

## How to Use
1. Download all three files.
2. Keep them in the same folder.
3. Open **index.html** in a browser.

## Features
- Clean layout
- Fully responsive
- Blue color theme
- Lightweight and easy to edit
